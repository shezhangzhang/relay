Foundational system components for Relay’s services.

Actors require an actix system to run. The system can be started using the Controller actor, which will also listen for shutdown signals and trigger a graceful shutdown. Note that actors must implement a handler for the Shutdown message and register with the controller to receive this signal. See the struct level documentation for more information.

See the Controller struct for more information.
